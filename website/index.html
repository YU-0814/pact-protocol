Real-World Use Case

PACT for Biomedical Data

Clinical trials, drug interactions, genomic data — structured data that AI agents need to consume precisely.

Clinical Trial Registry

A pharmaceutical database needs to serve clinical trial data to AI medical assistants. Traditional HTML pages waste tokens on navigation, ads, and formatting. PACT delivers just the data.

For biomedical convergence engineering, PACT enables:

  • Drug interaction databases for AI pharmacovigilance
  • Medical device catalogs with structured specs
  • Genomic variant portals with compression
  • Hospital API gateways for AI triage agents

Get Started

Quick Start

Add this file to /.well-known/pact.json and you are PACT-discoverable.

{
  "pact": "1.0",
  "site": "example.com",
  "description": "Your site description",
  "schemas": ["pact:commerce/product@1"],
  "endpoints": {
    "pact:commerce/product@1": {
      "list": "/pact/products",
      "item": "/pact/products/{id}",
      "search": "/pact/products?q={query}"
    }
  }
}
{
  "$pact": "1.0",
  "$s": "pact:commerce/product@1",
  "$t": 1710590400,
